DriveNotify Automator

The Problem

Teams often waste time manually checking shared folders for updates, or they miss critical document uploads (like signed contracts or invoices), leading to project delays.

The Solution

An n8n-based engine that uses the Google Drive API to "watch" for new files and immediately routes metadata (like the file name) into an automated SMTP mail server.

The Outcome

Eliminated manual folder monitoring, reduced response times to new file uploads to under 60 seconds, and ensured 100% notification reliability.
